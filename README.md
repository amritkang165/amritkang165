## ୨୧ Featured Projects

### ౨ৎ Study Companion
An AI-powered study platform that generates **summaries, flashcards, questions, and revision plans**.  
💖 Helped **100+ students** organize learning and automate revision cycles.  
**Tech:** React, Next.js, Tailwind CSS, Groq LLM API

### ౨ৎ LaunderLens
A graph-based **money muling and financial crime detection engine**.  
🔍 Processes **1,000+ transactions** and detects cycles, smurfing windows, and shell networks with graph insights.  
**Tech:** Next.js, React, Graph Algorithms, JSON Pipelines, Vercel

### ౨ৎ CrowdMind
A collective intelligence platform where users create **prediction questions**, submit forecasts with confidence, and build credibility over time based on accuracy.  
📈 Includes **binary, multiple choice, and probability-based prediction flows**, weighted consensus snapshots, authenticated workflows, and the foundation for rankings, trust recalculation, and live activity.  
**Tech:** React, Vite, Tailwind CSS, Zustand, Node.js, Express, PostgreSQL, WebSockets / Socket.io

### ౨ৎ Godot Games — Survival Mob & Sinking Ship
Built 2D survival/action game experiences with **enemy AI, scoring, traps, and dynamic difficulty**.  
🌟 Tested by **40+ players** with polished UI and smooth gameplay.  
**Tech:** Godot Engine, GDScript, Physics 2D, Tilemaps
